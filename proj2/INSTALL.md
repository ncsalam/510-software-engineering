## INSTALL.md

### Prerequisites

- Node.js 18+ and npm
- Ollama installed with your preferred model

### Steps

1. Clone the repository:
   ```bash
   git clone <repo-url>
   cd <repo-directory>
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the server:
   ```bash
   npm run start
   ```
4. Open browser at `http://localhost:8000`.
